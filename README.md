# Shaded
___
## What is Shaded?
Shaded is a community driven web application that will empower Vancouver residents to nominate, upvote, and track new tree planting locations in their neighborhood, helping the City prioritize where urban shade is needed the most.
___
## How to Run

**1. Download the files.**
**2. Open index.html in your browser.**
